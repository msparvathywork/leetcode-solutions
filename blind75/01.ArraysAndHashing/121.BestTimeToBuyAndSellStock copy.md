# Best Time to Buy and Sell Stock (#121)

## Problem Statement

You are given an array `prices` where `prices[i]` is the price of a stock on the `iᵗʰ` day.

You want to maximize your profit by choosing **one day to buy** and **one future day to sell**.

If no profit can be made, return `0`.

---

# Approach 1: Brute Force

## Idea

Try buying the stock on every day and selling it on every future day.

For every day:

- Assume today's price is the buying price.
- Check every future day as the selling price.
- Calculate the profit.
- Update the maximum profit.

---

## Algorithm

1. Iterate through each day as the buying day.
2. For each buying day, iterate through all future days.
3. Calculate:

```text
Profit = Selling Price - Buying Price
```

4. Update the maximum profit.
5. Return the maximum profit.

---

## Brute Force Solution

```python
class Solution(object):
    def maxProfit(self, prices):
        max_profit = 0
        n = len(prices)

        # Choose each day as the buying day
        for i in range(n):

            # Try selling on every future day
            for j in range(i + 1, n):

                profit = prices[j] - prices[i]

                if profit > max_profit:
                    max_profit = profit

        return max_profit
```

### Complexity Analysis

| Time      | Space    |
| --------- | -------- |
| **O(n²)** | **O(1)** |

# Approach 2: Optimized (One Pass)

## Optimized Approach (One Pass)

Instead of checking every possible buy-sell pair, we traverse the array only once.

While traversing:

- Keep track of the minimum buying price seen so far.
- If today's price is lower, update the buying price.
- Otherwise, calculate the profit if sold today.
- Update the maximum profit whenever a better profit is found.

---

## Algorithm

1. Initialize:

```python
buy = prices[0]
profit = 0
```

2. Traverse the array from the second element.

3. For every price:

- If the current price is less than the buying price,
  update the buying price.

```python
buy = prices[i]
```

- Otherwise, calculate today's profit.

```python
prices[i] - buy
```

- If the profit is greater than the current maximum profit,
  update the maximum profit.

## Code

```python
class Solution:
    def maxProfit(self, prices):
        buy = prices[0]
        profit = 0

        for i in range(1, len(prices)):

            if prices[i] < buy:
                buy = prices[i]

            elif prices[i] - buy > profit:
                profit = prices[i] - buy

        return profit
```

---

## Complexity Analysis

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| **O(n)**        | **O(1)**         |

---

## Why does this work?

Instead of asking:

> **"Which day should I buy?"**

We ask:

> **"What is the cheapest stock price I've seen so far?"**

Every new day becomes a potential selling day.

```
Current Price

↓

Profit = Current Price − Cheapest Price Seen So Far
```

This avoids checking every possible buy-sell pair and reduces the time complexity from **O(n²)** to **O(n)**.

---

## Key Takeaway

- Keep track of the **lowest buying price**.
- Calculate the profit if sold today.
- Store the maximum profit obtained.
- Only one traversal of the array is required.
